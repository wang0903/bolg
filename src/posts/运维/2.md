# Ubuntu 22.04 防火墙设置和开放端口命令

### 1、查看防火墙状态
```bash
sudo ufw status
```
### 2、开启防火墙
```bash
sudo ufw enable
```
### 3、关闭防火墙
```bash
sudo ufw disable
```
### 4、开放端口
```bash
sudo ufw allow 8080/tcp
```
### 5、查看已开放的端口
```bash
sudo ufw status numbered
```
### 6、关闭已开放的端口
```bash
sudo ufw delete allow 8080/tcp
``` 
### 7、查看防火墙日志
```bash
sudo ufw logging on
``` 
### 8、禁止防火墙日志
```bash 
sudo ufw logging off
```
